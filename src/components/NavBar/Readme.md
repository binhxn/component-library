NavBar can have one of three types:

- Tabs
- Underline
- Pill

```js
<NavBar >
  <NavBar.Item as='a' href='#navbar' type='tab'>
    Link 1
  </NavBar.Item>
  <NavBar.Item as='a' href='#navbar' type='tab'>
    Link 2
  </NavBar.Item>
  <NavBar.Item as='a' href='#navbar' type='tab'>
    Link 3
  </NavBar.Item>
</NavBar>

<NavBar>
  <NavBar.Item as='a' href='#navbar' type='underline'>
    Link 1
  </NavBar.Item>
  <NavBar.Item as='a' href='#navbar' type='underline'>
    Link 2
  </NavBar.Item>
  <NavBar.Item as='a' href='#navbar' type='underline'>
    Link 3
  </NavBar.Item>
</NavBar>

<NavBar>
  <NavBar.Item as='a' href='#navbar' type='pill'>
    Link 1
  </NavBar.Item>
  <NavBar.Item as='a' href='#navbar' type='pill'>
    Link 2
  </NavBar.Item>
  <NavBar.Item as='a' href='#navbar' type='pill'>
    Link 3
  </NavBar.Item>
</NavBar>

```
